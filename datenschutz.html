Grundsatz

VIRENNA verfolgt einen datensparsamen, werbefreien und ethikorientierten Ansatz. Die Nutzung erfolgt primär offline und ohne systematisches Tracking. Deine Daten gehören dir – sie werden niemals zu Werbezwecken verwendet oder verkauft.

Verarbeitung personenbezogener Daten

Diese Website speichert oder verarbeitet keine personenbezogenen Daten, solange du nicht aktiv Informationen per E-Mail übermittelst. Es werden keine Cookies gesetzt, keine Webfonts geladen und keine externen Analyse-Tools eingesetzt.

Die Website wird über GitHub Pages bereitgestellt. Dabei können durch GitHub aggregierte Zugriffsdaten (z. B. IP-Adressen in Logfiles) verarbeitet werden, auf die wir keinen direkten Zugriff haben. Es findet keine personenbezogene Auswertung statt.
